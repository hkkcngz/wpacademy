Tanışma ve Program Kurulumları

Git, Antigravity + Claude Eklentisi ve MAMP kurulumlarını gerçekleştiriyoruz.

Programa Genel Bakış

Bu bootcamp boyunca WordPress geliştirme ortamı kurulumundan başlayarak tema yapısı, eklenti ekosistemi, REST API ve performans optimizasyonuna kadar kapsamlı bir yolculuk yapacağız. İlk gün, verimli çalışabilmek için geliştirme araçlarımızı kuruyoruz.

1. Git Kurulumu

Git, kod değişikliklerini takip etmemizi ve projelerimizi yönetmemizi sağlayan versiyon kontrol sistemidir. WordPress geliştirme sürecinde temalar ve eklentiler üzerindeki değişiklikleri güvenle kayıt altına almak için kullanacağız.

1 İndirme ve Kurulum

git-scm.com/downloads adresine giderek işletim sisteminize uygun sürümü indirin ve kurulumu tamamlayın. Kurulum sihirbazında varsayılan seçenekleri bırakmak yeterlidir.

2 Kimlik Tanımlama

Kurulumun ardından terminal / komut istemcisini açın ve aşağıdaki komutlarla kendinizi tanıtın:

git config --global user.name "Adınız Soyadınız"
git config --global user.email "eposta@adresiniz.com"
3 Kurulumu Doğrulama
git --version
# Çıktı örneği: git version 2.x.x
💡 Windows kullanıcıları kurulum sırasında "Git Bash" seçeneğini aktif bırakırsa UNIX komutlarını da kullanabilir.

2. Antigravity Programı ve Claude Eklentisi Kurulumu

Antigravity, kod editörü olarak kullanacağımız geliştirme ortamıdır. Claude eklentisi ise yapay zeka destekli kod tamamlama ve açıklama özelliği sunar; bu sayede öğrenme sürecini hızlandırır.

1 Antigravity Kurulumu

Antigravity'nin resmi sitesinden güncel sürümü indirerek kurulumu tamamlayın. Kurulum sonrası programı ilk kez açtığınızda dil tercihi olarak Türkçe veya İngilizce seçebilirsiniz.

2 Claude Eklentisini Yükleme
  1. Antigravity'yi açın ve sol paneldeki Eklentiler (Extensions) ikonuna tıklayın.
  2. Arama kutusuna Claude yazın.
  3. Resmi Claude eklentisini bulup Yükle butonuna basın.
  4. Yükleme tamamlandıktan sonra editörü yeniden başlatın.
3 API Anahtarı Tanımlama

Claude eklentisini kullanabilmek için console.anthropic.com adresinden bir API anahtarı oluşturun. Anahtarı eklenti ayarlarına yapıştırarak bağlantıyı tamamlayın.

💡 API anahtarınızı .env dosyasında saklayın ve bu dosyayı asla Git'e eklemeyin.

3. MAMP Kurulumu

MAMP (Mac, Apache, MySQL, PHP), yerel bilgisayarınızda bir web sunucusu ortamı oluşturur. WordPress'i internet bağlantısı olmadan çalıştırıp test etmemizi sağlar.

1 İndirme

mamp.info/en/downloads adresinden işletim sisteminize uygun MAMP & MAMP PRO paketini indirin. (Ücretsiz MAMP sürümü yeterlidir.)

2 Kurulum ve İlk Başlatma
  1. İndirilen dosyayı çalıştırın ve kurulumu tamamlayın.
  2. MAMP'ı açın, Start Servers butonuna tıklayın.
  3. Apache ve MySQL göstergeleri yeşile döndüğünde sunucular hazır demektir.
3 Web Kök Dizini

WordPress dosyalarınızı aşağıdaki klasöre yerleştirin:

# macOS
/Applications/MAMP/htdocs/

# Windows
C:\MAMP\htdocs\
4 phpMyAdmin ile Veritabanı Oluşturma

MAMP çalışırken tarayıcıdan http://localhost:8888/phpMyAdmin adresine gidin. Sol panelden Yeni (New) seçeneğine tıklayarak wordpress_local adında yeni bir veritabanı oluşturun.

💡 MAMP varsayılan portu 8888'dir. Eğer 80 portunu kullanmak isterseniz MAMP > Tercihler > Portlar bölümünden değiştirebilirsiniz.
